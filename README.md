## ❗️ Troubleshooting

| Problem               | Solution                                       |
|-----------------------|------------------------------------------------|
| Asset sync failure    | Check your internet connection and relog       |
| Export error          | Clear application cache in settings            |
| UI lag                | Disable hardware acceleration or update GPU    |
| Security alert        | Add the application folder to exclusions       |
| Missing Pro features  | Ensure the Pro Resource profile is active      |
